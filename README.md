# TalentVision API

### Triagem Inteligente de Talentos com IA

#### Global Solution --- FIAP \| 2TDS \| 2025

## Visão Geral

O **TalentVision** é uma solução de triagem inteligente de talentos,
construída para a **Global Solution da FIAP**, cujo objetivo é
modernizar e automatizar a triagem de currículos entre empresas e trabalhadores.


#### Para acessar um endpoint vá para

    IP:8080 ou tv-app.azurewebsites.net/login

Agora acesse o login (admin / 123)

Estando logado agora em /resume/form
Você poderá enviar o seu Currículo para triagem e a platarma por meio de mensagens assincronas enviará uma mensagem para a sua empresa ideal!
